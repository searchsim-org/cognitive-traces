# Search & Interaction Datasets

A comprehensive collection of search, conversational, and recommendation datasets with real user interactions.

## Legend

* **Maintained?**: *Yes* (recent updates or active host); *Archive* (stable, no active updates); *Request* (gated by request form / approval); *Unknown* if unclear.
* **License/Terms** are summarized (exact text on resource page governs use).

### Top Picks for Search Research

1. **[TripClick](https://tripdatabase.github.io/tripclick/)** - Health search logs with real clicks + dwell time (research-only access, requires form)
   - 5.2M interactions from 2013–2020
   - Includes head/torso/tail splits for different query frequency analysis

2. **[ORCAS](https://msranorthamerica.github.io/ORCAS/)** - Bing clicks mapped to ClueWeb12
   - Easy to combine with TREC Deep Learning tracks
   - Large-scale click associations for learning-to-rank

3. **[MIMICS](https://arxiv.org/abs/2002.04284)** - SERP clarification panel interactions
   - Unique dataset for understanding clarification needs
   - Both Click and ClickExplore variants available

4. **[TREC Session Track (2010–2014)](https://trec.nist.gov/data.html)** - Classic multi-turn web search with clicks
   - Well-established benchmark
   - 5 years of session-based search data

5. **[TREC OpenSearch (2016–2017)](https://github.com/living-labs/trec-os-data)** - Live interleaving clicks on real sites
   - Open GitHub repository with detailed JSON format
   - CiteSeerX and SSOAR domains available

6. **[CWRCzech](https://github.com/seznam/CWRCzech)** - Modern, very large click+dwell dataset from Seznam.cz
   - 100M query-document pairs
   - 27.6M clicks with position and dwell time
   - Actively maintained

### For Conversational Search

- **[TREC CAsT](https://ir-datasets.com/)** - Multi-turn conversational search (2019, 2020 versions)
- **[TREC iKAT](https://ir.nist.gov/ikat/)** - Interactive knowledge assistance (2023)
- **[Qulac](https://github.com/aliannejadi/qulac)** / **[ClariQ](https://www.kaggle.com/datasets/konradb/chatbot-dataset-clariq)** - Clarifying questions and answers

### For E-Commerce

- **[SIGIR eCom Data Challenge 2021](https://arxiv.org/pdf/2104.09423)** - Large session log with queries, item clicks, and purchase data
